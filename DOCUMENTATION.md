## üìù Documentation Standards

### File Naming
- Main docs: `UPPERCASE.md` (e.g., README.md)
- Technical docs: `lowercase-with-dashes.md` (e.g., api-specifications.md)
- Component docs: Component folder + `README.md`

### Structure
- Clear headings with emojis
- Table of contents for long docs
- Code examples with syntax highlighting
- Cross-references between docs

### Updating Documentation
When updating code, also update:
- [ ] Relevant README files
- [ ] CHANGELOG.md
- [ ] API specifications (if API changed)
- [ ] Architecture docs (if structure changed)
